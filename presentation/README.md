# Luna Presentation

Professional HTML slideshow for video walkthrough.

## How to Use

1. **Open in browser:**
   ```bash
   open presentation/index.html
   # or
   firefox presentation/index.html
   ```

2. **Navigate:**
   - **Arrow Keys:** Left/Right to navigate
   - **Spacebar:** Next slide
   - **Buttons:** Click Previous/Next in bottom right

3. **Recording Tips:**
   - Open in full screen (F11)
   - Use 1920x1080 resolution for best quality
   - Navigate through slides while explaining each feature
   - Switch to live app demo at "Demo Time" slide

Edit `index.html` to:
- Change your name on title slide

- Add your GitHub URL
- Modify color scheme (CSS variables at top)
